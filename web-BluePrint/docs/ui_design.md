## 1. Design System Foundation

### 1.1 Design Principles

| Принцип | Описание |
|---------|----------|
| **Clarity First** | Складской работник в перчатках и при плохом освещении должен читать экран с первого взгляда |
| **Speed over Beauty** | Операция должна выполняться максимально быстро — минимум тапов, максимум контекста |
| **Error Resilience** | Система должна предотвращать ошибки, а не только их сообщать |
| **Offline Capable** | Мобильный интерфейс работает без сети — нет спиннеров, нет блокировок |
| **Density Control** | Веб-интерфейс плотный (много данных), мобильный — просторный (крупные кнопки) |

#### Варианты

```

│  [Удалить]      │  │  [Подтвердить]   │  │  [🔍]            │
│  bg: danger-500 │  │  bg: success-500 │  │  square, 36px    │
└─────────────────┘  └──────────────────┘  └──────────────────┘
```

#### Размеры

| Size | Height | Padding H | Font | Применение |
|------|--------|-----------|------|------------|
| `xs` | 28px | 10px | 12px | Таблицы, inline actions |
| `sm` | 32px | 12px | 13px | Card actions |
| `md` | 36px | 16px | 14px | Стандартный (default) |
| `lg` | 44px | 20px | 15px | Главные действия |
| `xl` | 52px | 24px | 16px | Mobile primary actions |
| `2xl` | 64px | 32px | 18px | ТСД — полноэкранные кнопки |

#### Состояния

```
Default  → bg: primary-500  text: white
Hover    → bg: primary-600  shadow: sm
Pressed  → bg: primary-700  scale: 0.98
Disabled → bg: neutral-200  text: neutral-400  cursor: not-allowed
Loading  → spinner + text  pointer-events: none
```

           ↓

    [Сканирует штрихкод товара]

      ┌────┴─────┐
      |           |
  [Товар         [Не найден]
   найден]           |
      |           [Ввести SKU вручную]
      ↓               |
  [Вводит кол-во] ←──┘
      |
      ↓
  [Сканирует ячейку размещения]
      |
    ┌─┴──────────────────────┐
    |                        |
[Ячейка подходит]    [Ячейка переполнена]
    |                        |
    ↓                [Система предлагает

    |
    ↓
[Подтверждает → inventory_movement создаётся]
    |

           |
           ↓
[Менеджер видит в Web: ПО завершён]
[Остатки обновлены]
```

```

## 9. Mobile UX для ТСД и Малых Экранов

### 9.1 Принципы адаптации для ТСД

| Аспект | Требование |
|--------|------------|
| **Touch Targets** | Минимум 48×48dp, рекомендуется 56×56dp для работы в перчатках |
| **Font Size** | Минимум 16sp для body, 20sp+ для ключевых данных |
| **Contrast** | WCAG AA minimum — 4.5:1 для текста; желательно AAA (7:1) для яркого света |
| **Gesture Area** | Нижняя 1/3 экрана — главные действия (thumb zone) |
| **One-Hand Operation** | Все ключевые элементы достижимы большим пальцем |
| **Scan-First UX** | Любое поле ввода имеет альтернативу через сканирование |
| **Haptic Feedback** | Подтверждение сканирования, успех/ошибка |
| **Audio Cues** | Звуковой сигнал на сканирование (настраивается) |

### 9.2 Адаптация под ТСД (Zebra TC21, 5" экран)

```
Экранная клавиатура ТСД имеет физические кнопки:
  - Scan trigger (боковая кнопка)
  - Enter
  - Backspace
  - Навигация (стрелки)

UI требования:
  - Все формы поддерживают Tab-навигацию
  - Enter = подтверждение действия
  - Scan trigger = открыть камеру
  - Escape = назад
  - F1-F4 = кастомные действия (настраивается)

Layout адаптации:
  - Кнопки confirm: BOTTOM, full-width, 56px height
  - Поля ввода: очень крупные (52px height)
  - Нет hover-состояний (нет мыши)
  - Цифровые поля: всегда открывать numpad
```

### 9.3 Accessibility

- `aria-label` на все иконки и icon-only кнопки
- `role="alert"` на toast уведомления
- `aria-live="polite"` для real-time обновлений количества
- Поддержка TalkBack (Android Accessibility)
- Цвета не единственный индикатор состояния (всегда + иконка или текст)

## 10. Animation & Motion Guidelines

### Принципы

| Принцип | Правило |
|---------|---------|
| **Functional** | Анимация несёт смысл (не декоративна) |
| **Fast** | Duration: 150–300ms. Длиннее — только для сложных переходов |
| **Easing** | ease-out для появлений, ease-in для исчезновений |
| **Reduced Motion** | `prefers-reduced-motion: reduce` — отключать все transition |

### Ключевые анимации

| Элемент | Анимация | Duration | Easing |
|---------|----------|----------|--------|
| Page transition | Fade + slide-up | 220ms | ease-out |
| Drawer open | Slide-in right | 280ms | cubic-bezier(0.32,0,0,1) |
| Modal open | Scale 0.95→1 + fade | 200ms | ease-out |
| Toast | Slide-in + fade | 180ms | ease-out |
| Scan success | Scale pulse + color | 250ms | spring |
| Button click | Scale 0.97 | 100ms | ease-in-out |
| Loading skeleton | Shimmer | 1200ms | linear (loop) |
| Number change | Count-up | 800ms | ease-out |
